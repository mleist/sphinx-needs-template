# Decisions

```{decision} Use organic compost instead of mineral fertiliser
:id: DEC_ORGANIC_FERT
:status: accepted
:tags: planting

We use organic compost for the vegetable and flower beds. Trade-off: slower
nutrient release in spring, but lower water-table contamination risk and
better long-term soil structure.
```

```{decision} Battery mower over petrol mower
:id: DEC_BATTERY_MOWER
:status: accepted
:tags: mowing

We adopt a cordless battery mower for the back-yard lawn. Trade-off:
shorter operating window per charge, but lower noise (helps with
NFR_NOISE_WINDOW) and no fuel storage at home.
```
