# Glossary

```{glossary_term} Mowing window
:id: G_MOWING_WINDOW
:tags: mowing

The legally and practically permissible time of day during which a power
lawn mower may be operated in a residential area. See `NFR_NOISE_WINDOW`.
```

```{glossary_term} Maturity days
:id: G_MATURITY_DAYS
:tags: planting

The expected number of days from planting to harvestability for a given
vegetable variety, as printed on seed packets. Used by
`days_until_harvest()`.
```

```{glossary_term} Patio
:id: G_PATIO
:tags: party_prep

The paved area immediately behind the house, used for outdoor dining and
hosting the small garden party.
```
